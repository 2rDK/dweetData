# dweetData
Send various data to Dweet.io

##Install
### RPi
1. First install PIP
```
apt-get install python3-pip
```
2. Install requests:
```
pip-3.2 install requests
```
3. Use **wget** to download raw repository form GitHub
4. Unzip package using **unzip** command
